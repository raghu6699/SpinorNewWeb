# Spinor Innovations Website

A modern, responsive website for Spinor Innovations, showcasing their services in website development, marketing, and app development.

## Services Offered

- **Website Development**: Custom websites tailored to your brand and business needs.
- **B2C Marketing**: Effective marketing strategies to engage and convert consumers.
- **B2B Marketing**: Targeted campaigns to connect with businesses and drive leads.
- **Retention Marketing**: Strategies to keep customers engaged and loyal to your brand.
- **App Development**: Innovative mobile and web applications for modern businesses.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the website.

## Technologies Used

- **Next.js**: React framework for production.
- **TypeScript**: Typed JavaScript for better development experience.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **ESLint**: Linting for code quality.

## Project Structure

- `src/app/`: Next.js app router pages and layouts.
- `public/`: Static assets.
- `src/app/globals.css`: Global styles with Tailwind.

## Deployment

This project can be deployed on Vercel, Netlify, or any platform supporting Next.js.

For Vercel deployment:

```bash
npm run build
```

Then deploy the `.next` folder.
